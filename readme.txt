A simple plugin for showing download button by shortcode.

== Description ==

This is a simple plugin for showing download button by shortcode.Use shortcode [download]your download link here[/download] to post or page,then see the result.

For live demo visit http://skjoybd.com

== Installation ==

<ul>
	<li>From your WordPress Dashboard Go for a "Add New" inside "Plugins" menu.</li>
	<li>Search for "JP Download Button Shortcode" and install the plugin.</li>
	<li>Activate the plugin</li>
</ul>

<strong>You:</strong> have finished your installation.<br>

<ul>
	<li>Now go to a post or page whatever you like.</li>
	<li>Paste this shortcode into the page or post and save it: <pre>[download]your download link here[/download]</pre></li>
	<li>Goto page or post preview,then see the result.</li>
</ul>
